| 1.  | +        | Addition      | x + y; Result = 8              |

| 3.  | x        | Multiplication| x * y; Result = 12             |
| 4.  | /        | Division      | x / y; Result = 3              |
| 5.  | %        | Modulus       | x % y; Result = 0              |
| 6.  | **       | Exponentiation| x ** y; Result = 36            |
| 7.  | ++       | Increment     | a = x++; Result = 9, x = 7     |
| 8.  | --       | Decrement     | c = x--; Result = 6, x = 5     |

## Page 12

# Bitwise Operators

## Bitwise Operators Explained

Bitwise operators are used to perform the operation at bits level.

Before a bitwise operation is performed, Javascript converts numbers to 32 bits signed integers and then performs the operation.

After the bitwise operation is performed, the result is converted back to 64 bits Javascript numbers.

---

| No. | Operator | Description | Example | Same as | Result (Binary) | Result (Decimal) |
|-----|----------|-------------|---------|---------|-----------------|------------------|
| 1.  | `&`      | Bitwise AND | 5 & 3   | 0101 & 0011 | 0001            | 1                |
| 2.  | `\|`     | Bitwise OR  | 4 \| 1  | 0100 \| 0001 | 0101            | 5                |
| 3.  | `~`      | NOT         | ~2      | ~0010     | 1101            | 13               |
| 4.  | `^`      | XOR         | 5 ^ 1   | 0101 ^ 0001 | 0100            | 4                |
| 5.  | `<<`     | Left Shift  | 4 << 1  | 0100 << 1  | 1000            | 8                |
| 6.  | `>>`     | Right Shift | 5 >> 1  | 0101 >> 1  | 0010            | 2                |

# Comparison Operators

## Comparison operators are used to compare two values and return true or false as a result.

---

| # | Operator | Operator Name        | Example (let x = 10) | Result |
|---|----------|----------------------|-----------------------|--------|
| 1 | `<`      | less than            | x < 10                | False  |
| 2 | `<=`     | less than or Equal to | x <= 10               | True   |
| 3 | `>`      | Greater than         | x > 5                 | True   |
| 4 | `>=`     | Greater than or equal to | x >= 5                | False  |
| 5 | `==`     | Equal to             | x == 5 x == 10        | False True |
| 6 | `!=`     | Not equal to         | x != 5                | True   |

Conditional statements are used to perform different actions based on different conditions.

In Javascript we have the following four Conditional statements:
1) `if` statement
2) `if...else` statement
3) `else if` statement
4) `switch` statement

Here are the Conditional statements in Javascript:

`if` statement: Used to execute a block of code if a condition is true.

```javascript
if (condition) {
  // code to be executed
}
```

## Understanding For Loops

The `for` loop consists of three expressions:

- The `for` loop consists of three expressions in it.

---

### First Expression: Initialization

- The first expression is called *initialization expression* and is used to check whether the loop is to be continued or not.

---

### Second Expression: Condition

- The second expression is called the *condition expression* and is used to check whether the loop is to be continued or not.

---

### Third Expression: Increment/Decrement

- The third expression is *increment/decrement*. It is used to change the index value for the next iterations.

---

### Syntax (General Form for Loop)

```javascript
for (expression 1; expression 2; expression 3) {
  statement 1;
  statement 2;
  ...
  statement N;
}
```

---

> **Pro-Tip:** Remember to initialize, check the condition, and increment/decrement within the `for` loop structure.
